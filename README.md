# ios12-calculator

使用vue cli3搭建的移动端项目，模仿iOS12系统下的IPhone计算器功能。

本项目适合vue新手入门参考，拷贝项目代码到本地后，输入命令`npm install`安装node_modules后，运行`npm run serve`就可启动项目。

涉及到的vue知识点有：

1. 列表渲染： v-for
2. Class 与 Style 绑定： v-bind:class; v-bind:style
3. 事件绑定: v-on
4. 计算属性： computed
5. 监听器： watch
